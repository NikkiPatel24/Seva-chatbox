How This AI Agent Works (Plain English)

1

User types a question

The visitor describes their situation in plain words — like "I need cheap blood pressure medication" or "where can I see a doctor for free in Cobb County?"

2

The agent reads & reasons

Their message is sent to Claude (an AI model made by Anthropic). Claude reads the full conversation history so it remembers everything said earlier in the same chat.

3

Claude picks the right resource

The agent has been given a detailed knowledge base of every page, clinic, pharmacy program, and county link on this website. It matches the user's need to the best option.

4

A helpful reply is shown

Claude writes a warm, clear response with the right links and any follow-up questions needed. The whole loop takes about 2–4 seconds and costs fractions of a cent per conversation.

5

The conversation continues

If the user asks a follow-up ("what about dental?"), the agent remembers the context and adjusts its answer without the user needing to re-explain their situation.

6

Zero personal data stored

The conversation lives only in the browser tab. When the user closes the page, nothing is saved. This protects visitor privacy and avoids HIPAA complications.
